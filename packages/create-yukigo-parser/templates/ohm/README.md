# Yukigo parser template (Ohm.js)

This template scaffolds a Yukigo parser package using **Ohm.js**.

## Included

- `src/index.ts`: parser entry point implementing `YukigoParser`
- `src/grammar.ohm`: Ohm grammar
- `tests/parser.spec.ts`: starter parser test

## Development

```bash
npm install
npm run build
npm test
```
