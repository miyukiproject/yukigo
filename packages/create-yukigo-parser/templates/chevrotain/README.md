# Yukigo parser template (Chevrotain)

This template scaffolds a Yukigo parser package using **Chevrotain**.

## Included

- `src/index.ts`: parser entry point implementing `YukigoParser`
- `src/parser.ts`: Chevrotain lexer and parser definitions
- `tests/parser.spec.ts`: starter parser test

## Development

```bash
npm install
npm run build
npm test
```
